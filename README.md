# Simulation-Modulation-2ASK-2PSK-2FSK-LabVIEW
Simulation des Modulations Numériques :Ce projet présente la simulation de trois types de modulations numériques binaires réalisée sous LabVIEW :-2ASK (Amplitude Shift Keying) : modulation par variation d'amplitude-2PSK(Phase Shift Keying) : modulation par variation de phase-2FSK (Frequency Shift Keying) : modulation par variation de fréquence
